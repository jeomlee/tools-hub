About this tool

Split PDF allows you to divide a PDF file into multiple smaller PDF files by page ranges. The tool runs entirely in your browser and does not upload your files. It is useful for extracting specific pages, separating chapters, or sharing only selected parts of a document.

All processing happens on your device. No files are uploaded.

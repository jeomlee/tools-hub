# Mini Tools Hub

A lightweight collection of browser-based utilities for PDF, image, and text tasks.

## PDF to JPG (No Upload)
I often need to convert PDF files into images without uploading sensitive documents.
This tool runs locally in your browser and supports JPG/PNG output.

Step-by-step guide:
https://tools-hub-4t9.pages.dev/guide/pdf-to-jpg.html

## Features
- PDF to JPG / PNG
- JPG to PDF
- Merge PDF
- Split PDF
- Word count
- Password generator
- Signature to PNG

## Privacy
Most tools process files locally in your browser.
No signup required.
